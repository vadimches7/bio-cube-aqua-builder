## Формат для генерации изображений

Для каждой рыбы нужно создать изображение с именем файла на основе:
- Русского названия (например: `Неон Голубой.jpg`)
- Латинского названия (например: `Paracheirodon innesi.jpg`)
- ID (например: `fish-1.jpg`)

Рекомендуемый формат: `{name_ru}.jpg` или `{name_lat}.jpg`
